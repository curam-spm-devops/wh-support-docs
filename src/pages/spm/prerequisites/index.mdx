Before you install or upgrade Merative™ Cúram, review the supported operating systems, prerequisites and the supported software to ensure compatibility.

A policy change has been implemented to the third-party prerequisites for Cúram version 7.0.0.0 and future versions of the product. Therefore, the prerequisites that are specified below now define support only for the third-party products for which the Cúram product has a direct dependency on. For more information about the policy change, see [here](https://www.merative.com/support/spm/news/social-program-management-third-party-prerequisites-policy-change).

The minimum hardware requirements for Cúram SPM development and test environments that were previously documented as part of the prerequisites are now documented in the Hardware requirements for [Cúram Social Program Management (SPM) Development and Test Environments tech note](https://www.merative.com/support/spm/question-answer/hardware-requirements-for-development-and-test1).
