## Supporting Materials​

Security Bulletins, system generated Tasks & Notifications documentation, HR1 Medicaid Community Engagement Cartridge supporting materials and Modern Java documentation are now available via the Merative Support Portal​.​


To access:​

- Log in to Merative Support
- Enter your credentials​
- Navigate to Knowledge Base, then Article Search, select "Curam Knowledge" as the
Data Category Group and then select the relevant Data Category.​

​
If you don’t yet have access, please submit a request through the portal.​
