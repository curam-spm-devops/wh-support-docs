## Cúram by Merative 8.2.2.0

Cúram by Merative™  is a leading solution for supporting the end-to-end social program and care coordination service delivery process. V8.2.2.0 includes enhancements to our technical infrastructure and user experience.


### Technical Infrastructure

The following technology version upgrades are supported in v8.2.2 keeping your deployments secure, compliant, and leveraging the latest technology standards. 
* Added support for **Oracle 26ai** 
* Introduced support **IBM Rational Software Architect (RSA) v10** with modern java version  
* Upgrade Carbon Design System to v11.97.0 
* Support for an F5 NGINX controller, as a replacement for Ingress NGINX controller 



### User Experience

#### Evidence Management  
Enhancements to the display of deleted evidence ensures that caseworkers get access to the evidence in all scenarios, supporting client queries. 


#### Accessibility – Caseworker and Clients    
Cúram is compliant with WCAG 2.1 level AA with exceptions. The 8.2.2 release continues our commitment to ensuring our systems meet recognized industry accessibility standards with additional enhancements across numerous WCAG 2.1 criteria to improve accessibility for both clients and caseworkers.  



### Workflow Efficiencies 
#### Task and Notification documentation 
Documentation on the business processes in Income Support solution that trigger Tasks and Notifications is now available from the  Knowledge Base section of the support portal.  

This provides customers with the information they need to decide which Tasks and Notifications they want to retain to best suit their requirements.  
 
For more details about these features, see the [What's new in 8.2.2.0](/pdf/spm/Curam_WhatsNew_822.pdf) and the [8.2.2.0 release notes](/pdf/releasenotes/curam8220.pdf).
